Unsupervised Ranking using Graph Structures and Node Attributes

Proposed an unsupervised ranking framework [1] exploiting both link structures and node attributes for better node importance ranking.

Leveraged two approximation tricks based on eigenvectors and Taylor expression to bring the time complexity of the algorithm linear to the number of nodes.

Provided theoretical justifications for the selection of hyperparameters under the unsupervised learning framework.
Boost Unsupervised Ranking via Link Discovery with Node Attributes

Proposed a joint learning scheme that incorporates link prediction (discovery) and node attributes (features) to boost unsupervised node ranking.

Designed a novel objective function integrating concepts of PageRank (ranking), degree-aware regularization, and link prediction with in-depth theoretical analyses.

Leveraged deep neural network (DNN) and multi-task learning to integrate link prediction, node attributes, and node ranking.
Survey of Matrix Factorization Model with Ratings and Features for Recommender Systems
